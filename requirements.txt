# External Tool Dependencies (Not pip)
# ============================================
# These are command-line tools that must be installed separately:
#
# CRITICAL (for full functionality):
# - enum4linux     → DCSeek enumeration
# - ldapsearch     → LDAPSeek queries (part of ldap-utils)
# - smbclient      → SMBSeek/ShareSeek share testing
# - nmap           → VulnSeek NSE scripts
# - snmpwalk       → SNMPSeek enumeration (part of snmp)
#
# OPTIONAL (enhanced features):
# - crackmapexec   → Advanced SMB/WinRM testing
# - impacket-*     → Kerberos attacks (GetNPUsers.py, GetUserSPNs.py)
# - rpcclient      → SMB RPC enumeration
# - nikto          → WebSeek vulnerability scanning
#
# ============================================
# Installation Commands by OS
# ============================================
#
# Kali Linux / Debian / Ubuntu:
#   sudo apt-get update
#   sudo apt-get install -y enum4linux ldap-utils smbclient nmap snmp-mibs-downloader
#   sudo apt-get install -y crackmapexec impacket-scripts nikto
#
# RHEL / CentOS / Fedora:
#   sudo yum install -y smbclient openldap-clients nmap net-snmp-utils
#
# macOS (Homebrew):
#   brew install samba openldap nmap net-snmp
#
# Windows (Chocolatey):
#   choco install nmap
#   # Most tools require WSL or manual installation from official sources
